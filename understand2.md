```
v-for遍历数组:value、index
v-for遍历对象:value、key、index
```
2、v-for中使用key达到单独渲染的目的<br>

3、变化响应式
```
变化数组的方法 => 原数据发生变化
替换数组 => 不影响原数组，只能通过赋值的方式让原数组发生变化
替换一个数据
针对对象的响应方法=>向对象添加一个新属性或者一些新属性
显示过滤/排序后的结果
```
